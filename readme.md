FavFood is a **full-stack food ordering web application** that allows users to browse food items, add them to a cart, place secure online orders, and track their order history.  
It also includes an **admin panel** for managing food items and customer orders.

This project is built to demonstrate **real-world application flow**, including authentication, payments, role-based access, and order management.
